quienmanda.es
=============

### Install locally

 Rails 2.0 needed. You can install it using RVM:
 
    $ rvm get stable
    $ rvm install 2.0.0
    
 Then install and run locally:
 
    $ git clone https://github.com/dcabo/quienmanda.es.git
    $ cd quienmanda.es
    $ bundle install

### Deploying in Heroku

    $ heroku apps:create
    $ git push heroku master
